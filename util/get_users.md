## Зачем

Если ботик подключился после создания группы, он не может анализировать участников ретроспективно. Чтобы был адекватный список неактивных участников, надо дать ему начальный список участников на текущий день.

### Подготовка

#### Регистрируем приложение

Зайди на my.telegram.org
API development tools
Создай приложение → получишь api_id и api_hash

#### Установи Telethon

```
python -m venv venv
source venv/bin/activate
pip install telethon
```

#### Запусти скрипт get_users.py

На компе скриптец запускается, скачивает пользователей чата в csv.
Ботику в личке надо задать команду /init_users и скормить содержимое этого csv
